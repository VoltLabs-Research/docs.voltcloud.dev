Whiteboards support freeform diagramming and note-taking within the team workspace.

![Whiteboards Overview](/assets/overviews/overviews_whiteboards.png)

## Listing and organization

The listing shows a thumbnail-first view of the boards in a team for visual identification alongside names.

![Whiteboard Listing](/assets/modules/whiteboards/whiteboard-listing.png)

## Canvas experience

The editor uses an infinite-canvas model with shapes, arrows, text, freehand drawing, and background controls.

## Real-time collaboration and sync behavior

Whiteboards synchronize board state in real time and show present collaborators. The conflict model is simple: when local and remote edits collide, the UI prompts to keep the local version or apply the remote one.

## Persistence and storage

Board state and supporting assets are stored on team infrastructure. Thumbnail previews in the listing are derived from that state.
