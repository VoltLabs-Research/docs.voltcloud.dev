Chat runs in the same team workspace as clusters, analyses, documents, and trajectories, supporting direct messages, group chats, presence, file sharing, reactions, and live updates.

![Chat Overview](/assets/overviews/overviews_chat.png)

## Sidebar and conversation flow

The sidebar organizes direct messages and group chats by recent activity. Opening a teammate's profile reuses the existing direct conversation rather than creating a duplicate.

## Group chats

Group chats scope conversations to a project or recurring topic. Admins manage membership, update group details, and assign other admins. Members can leave at any time.

## The real-time layer

Presence, typing indicators, unread state, reactions, and last-message updates stay synchronized across the sidebar and the active thread.

## Shared files and conversation context

Files travel alongside messages. The shared-files view collects every file exchanged in a conversation in one place, independent of the message history.
