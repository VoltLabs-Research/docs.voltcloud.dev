title: Chat
description: Real-time messaging for team conversations, group coordination, and shared files.
---

Chat operates in the same team workspace as clusters, analyses, documents, and trajectories. It supports direct messages, group chats, presence, file sharing, reactions, and live updates.

![Chat Overview](/assets/overviews/overviews_chat.png)

## Sidebar and conversation flow

The sidebar organizes direct messages and group chats by recent activity. Opening a teammate's profile reuses the existing direct conversation if one exists, rather than creating a duplicate.

## Group chats

Group chats hold conversations scoped to a project or recurring topic. Admins manage membership, update group details, and assign other admins. Members can leave a group at any time.

## The real-time layer

The real-time layer synchronizes presence, typing indicators, unread state, reactions, and last-message updates across the sidebar and the active thread.

## Shared files and conversation context

Chat carries files alongside messages. The shared-files view collects every file exchanged in a conversation in one place, independent of the message history.
