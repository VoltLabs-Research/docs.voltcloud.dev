## Overview

Volt AI is the built-in AI assistant that works across the entire VOLT platform. It supports **13 AI providers**, giving your team full flexibility to use the models and services that best fit your needs. Each team configures its own providers and API keys — there is no shared or centralized AI service.

Volt AI is not limited to a standalone chat interface. It is deeply integrated into other modules like the [LaTeX editor](/docs/modules/latex), where it can edit documents, compile projects, generate figures, fix errors, and more.

## Setting Up a Provider

To start using Volt AI, you need to add at least one AI provider to your team. Go to the **Volt AI** section from the sidebar and you will see the provider management panel.

![AI Providers](/assets/modules/team/ai-integration/providers.png)

Click the **Add Provider** button. You will be prompted to enter a name for the provider and your API key.

![Add Provider](/assets/modules/team/ai-integration/add-provider.png)

<Callout type="info">
API keys are encrypted before being stored. Each team manages its own keys — VOLT does not share keys across teams.
</Callout>

### Getting an API Key

Each provider has its own developer console where you can generate API keys. For example, here is the xAI console:

![xAI API Key](/assets/modules/team/ai-integration/xai-create-api-key.png)

### Selecting Models

After adding a provider, you can select which models to enable. Only enabled models will be available in the chat interface and AI-powered tools.

## Chat Interface

The chat interface supports real-time streaming responses, so you see the AI's reply as it is being generated. Conversations are saved per team and can be accessed by any team member.

![Conversation Example](/assets/modules/team/ai-integration/volt-ai-conversation-example-1.png)

Volt AI can interact with every module on the platform — from managing containers and trajectories to editing LaTeX documents and running analyses. It acts as a unified interface to your entire workspace.

![Conversation Example](/assets/modules/team/ai-integration/volt-ai-conversation-example-2.png)

## Task: Set up an AI provider

1. Go to the **Volt AI** section from the sidebar.
2. Click **Add Provider**.
3. Enter a name for the provider (e.g., "OpenAI Production").
4. Paste your API key into the key field.
5. Select the models you want to enable for your team.
6. Save the provider configuration.

**Expected outcome:** The provider appears in the provider list with its enabled models available in the chat interface and AI-powered tools.

## Task: Run an analysis using natural language

1. Open a **Volt AI** conversation from the chat interface.
2. Type a request such as "Run Coordination Analysis on trajectory X" (replacing X with the trajectory name).
3. Volt AI will identify the trajectory and the analysis plugin, then present a tool call for approval.
4. Review the parameters and click **Approve** to authorize execution.
5. Wait for the analysis job to start.

**Expected outcome:** The analysis appears in the Jobs panel as running, and once completed, the results are accessible from the trajectory's analysis listing.
