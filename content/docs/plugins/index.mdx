title: Plugins
description: Browse the analysis plugins published to the VOLT Registry, and learn how the plugin system works.
---

Plugins are node-based analysis workflows that run scientific binaries on your cluster and feed results back into VOLT as listings, charts, and 3D artifacts. Install any of them with [vpm](/docs/open-source/vpm) — for example `vpm install @voltlabs/opendxa`.

</Cards>

## Structural Analysis

<Cards>
  <Card title="Adaptive Common Neighbor Analysis" href="/docs/plugins/adaptive-common-neighbor-analysis" description="Run ACNA/CNA, label each atom by local structure, and emit the cluster-graph artifacts consumed by OpenDXA." />
  <Card title="Polyhedral Template Matching" href="/docs/plugins/polyhedral-template-matching" description="Classify atoms by polyhedral template (FCC, BCC, HCP, SC, diamond) and export structure type plus cluster-graph tables." />
  <Card title="Pattern Structure Matching" href="/docs/plugins/pattern-structure-matcher" description="Match local environments against user-defined reference patterns and export a reconstructed-state contract for OpenDXA." />
  <Card title="Ackland-Jones Analysis" href="/docs/plugins/ackland-jones" description="Identify FCC, HCP, BCC, and ICO via adaptive bond-angle analysis — no cutoff parameter required." />
  <Card title="Identify Diamond Structure" href="/docs/plugins/identify-diamond" description="Detect cubic and hexagonal diamond using second-neighbor-shell CNA, labelling core and neighbour-shell atoms." />
  <Card title="Chill+ Ice & Hydrate Classifier" href="/docs/plugins/chill-plus" description="Classify hexagonal, cubic, and interfacial ice plus hydrate phases via Chill+ bond-order correlation." />
  <Card title="Steinhardt Order Parameters" href="/docs/plugins/steinhardt-order-parameters" description="Compute bond-orientational order parameters q_l and Wigner invariants per atom for crystallinity and nucleation studies." />
</Cards>
